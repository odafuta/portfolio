## 1. 設計概要

### 1.1 デザインコンセプト

#### **コアメッセージ**
```
"技術力と人間性を両立したIT人材"
- 論理的思考と創造性のバランス
- 継続的学習と成長への意欲
- チームワークとコミュニケーション能力
```

#### **デザインテーマ**
- **プロフェッショナル**: 企業採用担当者が安心できる品質
- **親しみやすさ**: 個人写真で人柄をアピール
- **技術的洗練**: モダンな技術スタックの適用
- **成長性**: 学習過程と将来ビジョンの可視化

### 1.2 ターゲットユーザー分析

#### **プライマリユーザー: IT企業採用担当者**
```markdown
# ユーザージャーニー
1. **認知段階**: LinkedIn/GitHub/求人サイトからの流入
2. **興味段階**: ファーストビューで技術力と人柄を判断
3. **検討段階**: プロジェクト詳細で技術的深度を評価
4. **決定段階**: 面接招待の意思決定

# 期待値
- 3分以内での候補者評価完了
- 技術レベルの客観的判断材料
- 面接での質問項目の発見
- チームフィット感の事前確認
```

### 1.3 成功指標

#### **定量指標**
- セッション時間: 平均3分以上
- 直帰率: 30%以下
- プロジェクト詳細閲覧率: 60%以上
- コンタクトフォーム到達率: 15%以上

#### **定性指標**
- 技術力の適切な伝達
- 人格的魅力の表現
- プロフェッショナリズムの印象
- 成長ポテンシャルの訴求

// useScrollAnimation.ts
export const useScrollAnimation = () => {
  useEffect(() => {
    const observer = new IntersectionObserver(
      (entries) => {
        entries.forEach((entry) => {
          if (entry.isIntersecting) {
            entry.target.classList.add('animate-in');
          }
        });
      },
      {
        threshold: 0.2,
        rootMargin: '0px 0px -50px 0px'
      }
    );

    const elements = document.querySelectorAll('[data-animate]');
    elements.forEach((el) => observer.observe(el));

    return () => observer.disconnect();
  }, []);
};

### 設計のハイライト

#### **1. 写真スライダーの戦略的活用**

- **What**: 成果の定量化と価値創造

- **拡張性**: 新しいプロジェクト追加の容易性
- **メンテナビリティ**: 長期運用を考慮した構造

この設計に基づいて実装することで、技術力・人間性・成長ポテンシャルを効果的にアピールできるポートフォリオサイトが完成します。
